# tmix-icon-rails

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tmix-icon-rails'
```

And then execute:
```sh
$ bundle
```

Or install it yourself as:
```sh
$ gem install tmix-icon-rails
```

## Usage

```css
@import "tmix-icon"
```

## Do you need list of icon and css class name?
```sh
$ ruby test/gen_table.rb > list_of_icons.html
```
